En este ejemplo se empleó el lenguaje de programacion Python para desarrollar una aplicacion web, que permite sustraer datos de paginas en internet que no esten creadas con JavaScript.

Python utilizas dos tecnicas muy polulares para lograr el objetivo anteriormente planteado. las cuales tienen por nombre: Beautiful Soup 4 y Scrapy.

¿Qué es Scrapy?

Scrapy es una plataforma colaborativa o framework de código libre que corre en el lenguaje de Programación Python, cuya finalidad es extraer datos de paginas webs, para ser utilizados para estudios como: La minería de datos, procesamiento de informacíon, manejo de registros históricos, entre otros posibles usos.

Scrapy tiene las siguientes caracteristicas:

  • Rápida y poderosa: Escribes las reglas para extraer los datos y Scrapy hace el resto.
  • Fácilmente extensible: Dada su configuración, esta puede generar nuvas funcionalidades sin tener que modificar el código fuente.
  • Portable y Pythonico: Esta caracteristica hace referencia a qu está escrito en Python y puede correr bajo Linux, Windows, Mac y BSD.

Paso a paso para realizar la tecnica Scraping: Para implementar esta tecnica de recolección de informacion en la web, se requiere de las siguientes librerias de python.


  • La librería Requests: Provee de metodós para obtener todo tipo de información de una URL, ademas nos sirve para normalizar datos en tipo JSON.

  • La librería Beautiful Soup 4: Es una librería que permite extraer datos de paginas HTML.

SCRAPING WEB:

Es una técnica utilizada mediante programas de software para extraer información de sitios web.
